|_.Тип|_.Описание|
|string|Строка. Если набор значений ограничен, то следом приводится наименование перечисления (пример: string Status)|
|number|Число. Если набор значений ограничен, то следом приводится наименование перечисления (пример: number Level)|
|boolean|Булево значение (true/false)|
|object|Ассоциативный массив. Наименование структуры приводится следом (пример: object Params)|
|array|Список. Тип приводится следом (примеры: array string, array object Params)|
|nothing|Атрибут отсутствует|
