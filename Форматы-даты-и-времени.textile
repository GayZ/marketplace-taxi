Все форматы соответствуют стандарту "ISO 8601":https://ru.wikipedia.org/wiki/ISO_8601.

h3. string OffsetDateTime

Формат: @YYYY-MM-DDThh:mm:ss±hh:mm@

h3. string LocalDateTime

Формат: @YYYY-MM-DDThh:mm:ss@

h3. string Duration

Формат: @PnYnMnDTnHnMnS@

Если @n@ равно нулю, указывать не обязательно.

Пример: @PT35M13.106S@

Раздел в википедии: "ISO 8601: Durations":https://en.wikipedia.org/wiki/ISO_8601#Durations
